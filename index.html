Mission Statement

Housemates is a platform with which roommates can connect quickly and concisely. Using the features implemented on the webpage, roommates can easily reference a shared calendar, view a consistent shopping list, effectively divide chores and tasks, and easily change landlord or contact information for the entire household. This is done using a household ID, which can track every roommate within the household and ensure that everyone has access to the information that they need.
